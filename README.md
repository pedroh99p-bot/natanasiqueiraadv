# Landing Page Dr. Natan Siqueira

Landing page mobile-first para advocacia criminal, construída com Astro e CSS com tokens.

## Requisitos

- Node.js 20+

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Verificações

```bash
npm run check
npm run build
```

## Pré-visualização

```bash
npm run preview
```

## Deploy

O projeto é estático e pode ser publicado em Vercel a partir da branch `main`.

## Observações

- WhatsApp e Instagram estão configurados.
- LinkedIn, e-mail e número completo da OAB continuam pendentes e não são renderizados como links vazios.
